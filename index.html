Showpony is a multimedia engine for the web! Version: DEV

Setup

Download the latest release and put the showpony folder files onto your website! Then use the following code:

HTML

<link rel="stylesheet" href="showpony/styles.css">
<script src="showpony/script.js"></script>
<script src="https://heybard.com/apis/accounts/script.js"></script>

JavaScript

new Showpony;

You can pass many options to the Showpony object, which are covered in the Wiki.
